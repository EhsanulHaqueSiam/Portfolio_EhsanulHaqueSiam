# CVPR (Computer Vision Pattern Recognition)

**GitHub:** https://github.com/EhsanulHaqueSiam/CVPR  
**Type:** Academic/Research | **Language:** Python (Jupyter Notebooks)

---

## Overview
A collection of advanced computer vision and machine learning projects implemented **from scratch** without high-level ML libraries (scikit-learn, TensorFlow). Demonstrates deep understanding of mathematical and architectural principles behind computer vision algorithms.

---

## Projects

### 1. KNN Image Classification (Assignment 1)

| Aspect | Details |
|--------|---------|
| **Goal** | Compare distance metrics for image classification |
| **Dataset** | Animal Dataset (cats, dogs, pandas) |
| **Algorithm** | K-Nearest Neighbors (from scratch) |
| **Techniques** | L1/L2 distance comparison, 5-Fold Cross-Validation |
| **Preprocessing** | Grayscale conversion, normalization, flattening |

### 2. Multi-Class Neural Network (Assignment 2)

| Aspect | Details |
|--------|---------|
| **Goal** | 5-class classification with deep network |
| **Architecture** | 3 hidden layers (NumPy only) |
| **Activations** | ReLU (hidden), Softmax (output) |
| **Loss** | Categorical Cross-Entropy |
| **Init Method** | He Initialization |
| **Accuracy** | **~96%** |
| **Metrics** | Precision, Recall, F1-Score, Confusion Matrix |

## Key Implementations (From Scratch)

- K-Nearest Neighbors algorithm
- Manhattan (L1) and Euclidean (L2) distance metrics
- K-Fold Cross-Validation
- Multi-layer Perceptron with backpropagation
- He Initialization for weight initialization
- ReLU and Softmax activation functions
- Categorical Cross-Entropy loss
- Precision, Recall, F1-Score metrics
- Confusion Matrix visualization
