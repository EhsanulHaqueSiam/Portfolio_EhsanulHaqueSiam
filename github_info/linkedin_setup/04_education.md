## üí° Education Section Tips

### School Selection
- Make sure to select the official LinkedIn pages of your schools
- This connects you to alumni networks
- Shows up in "People You May Know"

### Dates
- If still studying, use "Expected 2026"
- Include month if you remember (e.g., "Jan 2022 - Dec 2026")

### Grade
- Add GPA/CGPA if it's above 3.5/4.0
- For percentage, add if above 75%
- Skip if not impressive

### Activities
- Focus on technical/relevant activities
- Include leadership roles
- Mention competitions and wins
- Add any student organizations
